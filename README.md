Install the following globally:
- node && npm
- bower
- gulp
- [coffeegulp] (https://github.com/minibikini/coffeegulp)

Satisfy Project Dependencies:

```
npm install
bower install
```

Run gulp via coffeegulp, ```coffeegulp``` or shell script included ```cgulp```


tasks:

- serve or s
- clean
- vendor_develop
- vendor_prod
- default or "empty"
